# Security Policy

## Supported Versions
Only the latest published version of **bifurcated_sort** receives fixes or updates.

| Version | Supported |
|--------|-----------|
| Latest | ✔️ |
| Older  | ❌ |

## Reporting a Vulnerability
If you discover a security issue, please open an issue here:

https://github.com/balajisuresh1359/bifurcated_sort/issues

Please include:
- A clear description of the problem  
- Steps to reproduce  
- Expected vs actual behavior  

I will review valid reports and provide a fix or response as soon as possible.
